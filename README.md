# my-first-project

연습용으로 사용할 수 있는 아주 간단한 파이썬 프로그램입니다.

## 폴더 구조

```
.
├── README.md
└── main.py
```

## 프로그램 설명

`main.py`는 사용자의 이름을 입력받아 친절한 인사 메시지를 출력하는 예제입니다.
입력이 비어 있으면 기본값으로 "친구"를 사용합니다.

## 실행 방법

1. 파이썬 3이 설치되어 있는지 확인합니다.
2. 터미널에서 프로젝트 폴더로 이동합니다.
3. 아래 명령을 실행합니다.

```
python3 main.py
```

실행 후 이름을 입력하면 인사 메시지가 출력됩니다.
